Chaos Games

Drawing fractals one random point at a time

A chaos game is a relatively simple iterative process. Given a shape and a ratio, choose an initial position. Now, pick a random vertex from this shape. The next position is found by drawing a vector from from the current position to the randomly chosen vertex and scaling that vector by the ratio. Recurse.

We can write this using a recurrence relation. Let v[n] represent the output of a random process where the nth value is a random vertex from our shape (assume vertices and points are represented by vectors obeying normal vector math):

p[0] = p0
p[n+1] = p[n] + r*(v[n] - p[n])
